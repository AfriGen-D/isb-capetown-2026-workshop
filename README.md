## Overview

The African Genomics Data Hub (AfriGen-D) provides
essential curated resources and tools specifically
designed for analyzing African genetic data, addressing
unique challenges in variant discovery, imputation
accuracy, and knowledge integration.

Participants engage with real African genomic datasets
through practical exercises, learning to navigate the
complete biocuration pipeline from quality control
through variant annotation. The workshop demonstrates
how curated, population-specific reference panels and
specialized databases deliver **15--40% accuracy
improvements** over generic reference panels.

## Resources Covered

- **AfriGen-D Imputation Service** -- web-based
  genotype imputation with African-specific reference
  panels. See <https://impute.afrigen-d.org>
- **African Genomics Medicine Portal (AGMP)** --
  2000+ curated pharmacogenomic variants
- **African Genomics Variation Database (AGVD)** --
  population-specific allele frequency analysis
