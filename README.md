# ISB Cape Town 2026 -- Workshop Hub

Workshop materials for the **19th Annual International
Biocuration Conference**, Cape Town (April 2026), hosted
at the African Bioinformatics Institute (ABI),
University of Cape Town.

**View Online:** <https://afrigen-d.github.io/isb-capetown-2026-workshop/>

## Overview

This repository is the landing hub for the twelve
workshops held at ISB Cape Town 2026. Each workshop
entry links to its own tutorial materials, data, and
hands-on exercises.

## Workshops

1. Programming / Coding / Linux
2. AI-assisted Workflow
3. Genome Assemblies
4. Research Data Management
5. Foundations of Biocuration
6. Metadata Harmonisation
7. FAIR Principles
8. Spatial Transcriptomics
9. GWAS
10. Data Stewardship
11. Biocuration Workflows
12. Data Sharing

## Participants

~83 registered participants from 29 countries; 21 paid
in-person attendees. See the hub page for workshop
assignments and pre-workshop skill expectations.
