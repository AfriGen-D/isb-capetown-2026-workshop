# AfriGen-D Imputation Tutorial -- ISB Cape Town 2026

Tutorial materials for the workshop **"Genotype Imputation
and Data Analysis for African Populations: A Hands-on
Tutorial Using AfriGen-D's Curated African Genomic
Resources"**, held at the **19th Annual International
Biocuration Conference (ISB Cape Town 2026)**,
African Bioinformatics Institute, University of Cape
Town -- Monday, 20 April 2026, 14:00--18:00 SAST.

**View Tutorial Online:** <https://afrigen-d.github.io/isb-capetown-2026-workshop/>

## Overview

This repository contains a hands-on tutorial for using
the [AfriGen-D Imputation Service][impute], a free
web-based platform for African genotype imputation.

Originally developed for the 1st African Genomics Short
Course (December 2025), adapted and extended for the
2026 ISB Cape Town biocuration workshop.

[impute]: https://impute.afrigen-d.org

## Contents

- `Tutorial-AfriGenD-Imputation-Service.md` -- Complete
  tutorial guide
- `data/sparse_peaks_dense.vcf.gz` -- Sample genotype
  data (VCF format, chromosome 22)
- `data/1k_afr_661_samples_phenotype.txt` -- Sample
  phenotype file for GWAS

## What You'll Learn

1. Creating an account and navigating the platform
2. Running genotype imputation with the H3Africa V6HC-S
   reference panel
3. Using the Allele Switch Checker for quality control
4. Performing VCF Liftover between genome builds
5. Running GWAS analysis on sparse vs imputed data
6. Interpreting imputation quality metrics and GWAS
   results

## Requirements
